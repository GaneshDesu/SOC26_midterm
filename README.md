# SOC-26 Project – Competitive Programming

## About the Project

This project was a part of the Summer of Code program where the main focus was on building a strong foundation in **Competitive Programming** and improving problem-solving skills. Throughout the project, I revised important programming concepts in C++ and gradually explored various data structures, algorithms, and problem-solving techniques that are commonly used in coding interviews and competitive programming contests.

The project was structured week by week, with each week focusing on a different set of concepts and practice problems.


## Learnings from the Project

### Week 1 – Building C++ Fundamentals

During the first week, the main focus was on strengthening the basics of C++ and getting comfortable with commonly used language features.

Topics covered:

* Revised fundamental **C++ syntax and programming concepts**
* Practiced using **STL Containers** such as:

  * Vector
  * Array
* Learned and implemented basic data structures:

  * Stack
  * Queue
  * Linked List
* Practiced important problem-solving techniques such as:

  * Two Pointer Technique
  * Sliding Window Technique
* Solved multiple beginner-level programming problems for practice

This week helped in refreshing programming fundamentals and understanding how commonly used STL containers work internally and in practice.


### Week 2 – Sorting and Searching Algorithms

The second week focused on understanding some of the most important algorithms used frequently in programming.

Topics covered:

* Learned and implemented basic sorting algorithms:

  * Bubble Sort
  * Merge Sort
* Learned searching techniques:

  * Linear Search
  * Binary Search
* Explored useful built-in STL algorithms such as:

  * sort()
  * binary_search()
* Studied and compared the **time complexities** of different sorting and searching algorithms
* Practiced several problems based on searching and sorting concepts

This week helped me understand how algorithm efficiency affects performance and when to choose one algorithm over another.


### Week 3 – Greedy Algorithms and Recursion

This week focused more on developing problem-solving intuition and understanding algorithm design strategies.

Topics covered:

* Learned the concept of **Greedy Algorithms**
* Understood situations where greedy strategies can and cannot be applied
* Solved problems involving greedy decision making
* Learned about **Recursion** and recursive problem solving
* Studied **Memoization** as an optimization technique for recursive solutions
* Practiced problems based on recursion and greedy approaches

This week improved my understanding of breaking problems into smaller subproblems and thinking about optimal choices while solving problems.


### Week 4 – Trees and Dynamic Programming

The final week covered advanced data structures and algorithmic concepts that are very important in competitive programming.

Topics covered:

* Learned the fundamentals of **Trees** and **Binary Search Trees (BST)**
* Understood tree traversal techniques:

  * Breadth First Search (BFS)
  * Depth First Search (DFS)
* Learned the basics of **Dynamic Programming**
* Understood how DP helps optimize overlapping subproblems
* Practiced problems related to:

  * Trees
  * Graph Traversals
  * Dynamic Programming

This week introduced me to advanced problem-solving techniques that are widely used in coding contests and technical interviews.


## Overall Takeaways

Through this project, I was able to strengthen my programming fundamentals and significantly improve my approach to problem solving.

Key takeaways from the project:

* Improved proficiency in **C++ programming**
* Developed better understanding of **STL and common data structures**
* Learned several important **algorithmic techniques**
* Built stronger problem-solving intuition through consistent practice
* Gained exposure to advanced topics like **Trees, Graph Traversals, Greedy Algorithms, and Dynamic Programming**
* Developed a more structured approach towards solving competitive programming problems

This project helped me build a strong foundation for further practice in competitive programming and coding interviews.
